Найди и исправь ошибки TypeScript в указанном файле.

Входные данные:
$ARGUMENTS

Формат аргументов:
- Абсолютный или относительный путь к файлу

Шаги:
1. Прочитай файл
2. Запусти `pnpm tsc --noEmit` и отфильтруй ошибки для указанного файла
3. Проанализируй каждую ошибку
4. Исправь ошибки, сохраняя логику кода
5. Повторно проверь типы после исправлений

Пример:
```
/tsfix packages/core/modules/router.ts
```
