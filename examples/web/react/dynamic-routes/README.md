# React Dynamic Routes Example

Demonstrates runtime route management with feature flags.

## What it covers

- `getRoutesApi(router).add()` — add routes (with nested children) at runtime
- `getRoutesApi(router).remove()` — remove routes at runtime
- Sidebar updates dynamically when routes are added/removed
- Redirect to Home when active route is removed

## Run

```bash
pnpm install
pnpm dev
```
